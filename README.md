# Little Stars Crèche

A single-page landing website for Little Stars Crèche, a warm and nurturing childcare facility. This site provides prospective families with essential information about our services, environment, and how to get in touch.

## Project Description

Little Stars Crèche is a thoughtfully designed single-page landing site that showcases our childcare facility's welcoming atmosphere and professional services. The website emphasizes trust, warmth, and transparency, helping parents make informed decisions about their child's care.

## Getting Started

### Local Development

There are two ways to view the site locally:

1. **Direct File Opening:**
   Simply open `index.html` in your web browser by double-clicking the file or dragging it into a browser window.

2. **Local Server (Recommended):**
   For a more accurate development environment, run a local HTTP server:
   ```bash
   python3 -m http.server
   ```
   Then navigate to `http://localhost:8000` in your browser.

## Deployment

This site is configured for GitHub Pages deployment and serves directly from the `main` branch. Once pushed to GitHub, enable GitHub Pages in your repository settings to publish the site.

## Tech Stack

- **HTML5:** Semantic markup for accessibility and SEO
- **CSS3:** Modern CSS with custom properties and responsive design
- **Vanilla JavaScript:** Pure JavaScript for interactions (no frameworks)
- **Google Fonts:** Nunito font family for warm, friendly typography
- **Unsplash Images:** High-quality photography showcasing childcare environments

## Features

- Fully responsive design for mobile, tablet, and desktop
- Accessibility-first approach with semantic HTML
- Smooth scroll animations and interactive elements
- Optimized performance with lazy-loading images
- SEO-friendly structure with proper meta tags
